# Task Category User Stories

## US-CAT-001: Create Task Category

**As a** household Owner or Admin (depending on hierarchy)
**I want to** create a task category
**So that** I can organize tasks into groups

### Acceptance Criteria
- Category has a name
- Optional color can be set
- Sort order can be specified

---

## US-CAT-002: List Task Categories

**As a** household member
**I want to** see all categories in the household
**So that** I can filter and organize tasks

### Acceptance Criteria
- Returns all categories
- Shows name, color, and sort order

---

## US-CAT-003: View Category Details

**As a** household member
**I want to** view a specific category
**So that** I can see its properties

### Acceptance Criteria
- Shows category name
- Shows color if set
- Shows sort order

---

## US-CAT-004: Update Category

**As a** household Owner or Admin
**I want to** modify a category
**So that** I can rename or reorder it

### Acceptance Criteria
- Can change name
- Can change color
- Can change sort order

---

## US-CAT-005: Delete Category

**As a** household Owner or Admin
**I want to** remove a category
**So that** unused categories are cleaned up

### Acceptance Criteria
- Category is removed
- Tasks in that category have their category cleared
